Full Course Curriculum

 
Note: Each Unit takes around one to two classes to complete 

   
 
├── Scratch
│   ├── Unit 1: Getting Started
│   │   ├── Basic knowledge of Scratch's building blocks
│   │   ├── Understanding Scratch mechanics (events, sequences, loops)
│   │   ├── Introduction to the Scratch editor and sprites
│   │   ├── Basic understanding of costumes and drawing tools
│   │   └── Project: Create a basic interactive animation
│
│   ├── Unit 2: Making Things Move
│   │   ├── Moving objects and changing costumes
│   │   ├── Using keypresses and mouse input
│   │   ├── Adding sound and reactions to events
│   │   └── Project: Make a simple character that moves and reacts
│
│   ├── Unit 3: Logic and Variables
│   │   ├── Understanding if-then and repeat loops
│   │   ├── Using variables to track score, speed, and lives
│   │   ├── Creating and using simple functions (My Blocks)
│   │   └── Project: Build a clicker game or reaction timer
│
│   ├── Unit 4: Levels and Interactions
│   │   ├── Detecting collisions between sprites
│   │   ├── Broadcasting messages and syncing sprite behavior
│   │   ├── Switching backgrounds and creating scenes
│   │   └── Project: A maze game with multiple levels
│
│   ├── Unit 5: Game Design Essentials
│   │   ├── Game loop structure in Scratch
│   │   ├── Lives, health bars, and power-ups
│   │   ├── Timer and scorekeeping mechanics
│   │   └── Project: Build a survival game with a time limit
│
│   ├── Unit 6: Platformer Prep
│   │   ├── Gravity simulation with change y by blocks
│   │   ├── Ground detection and platforms
│   │   └── Project: Mini jump-and-land challenge
│
│   └── Unit 7: Building a Platformer Game
│       ├── Character physics: jumping, falling, landing
│       ├── Adding enemies and moving obstacles
│       ├── Creating levels with checkpoints
│       ├── Win/loss conditions and menus
│       └── Final Project: Full-featured platformer game

 

├── Python
│   ├── Unit 1: Getting Started
│   │   ├── Installing Python and using online editors (like Replit)
│   │   ├── Writing your first program: print("Hello, world!")
│   │   ├── Understanding strings, numbers, and comments
│   │   └── Project: Make a personal greeting program
│
│   ├── Unit 2: Variables and Input
│   │   ├── Using variables to store information
│   │   ├── Taking input from the user
│   │   ├── Converting between strings and numbers (int, str)
│   │   └── Project: Create a basic calculator
│
│   ├── Unit 3: Logic and Decisions
│   │   ├── If/else statements and comparison operators
│   │   ├── Using Boolean logic (and, or, not)
│   │   └── Project: Make a number guessing game
│
│   ├── Unit 4: Loops and Repetition
│   │   ├── While loops and for loops
│   │   ├── Using `range()` and loop counters
│   │   ├── Breaking out of loops with `break`
│   │   └── Project: Make a password checker or timer
│
│   ├── Unit 5: Lists and Data
│   │   ├── Creating and accessing lists
│   │   ├── Looping through lists
│   │   ├── Adding, removing, and sorting items
│   │   └── Project: To-do list or favorite movies tracker
│
│   ├── Unit 6: Functions and Modules
│   │   ├── Defining and calling functions
│   │   ├── Using parameters and return values
│   │   ├── Importing standard Python modules (like `random` and `time`)
│   │   └── Project: Rock-paper-scissors game using functions
│
│   └── Unit 7: Final Project – Text Adventure Game
│       ├── Designing rooms and choices using dictionaries/lists
│       ├── Using functions to organize game code
│       ├── Adding inventory and player stats
│       ├── Handling win/lose conditions
│       └── Final Project: Build your own branching-story game!
